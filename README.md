# 📰 GECA News Updates

**GECA News Updates** is a web application developed for **Government College of Engineering, Aurangabad (GECA)** to publish and manage official campus news and announcements.  
It automates news updates from official sources and displays them through a modern, responsive, and interactive interface.

---

## 🌐 Live Link
🔗 [https://gecaupdates.onrender.com/](https://gecaupdates.onrender.com/)

---

## 🧩 Tech Stack
- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Tools & Libraries:** Cheerio, Cron, Nodemailer, JWT  

---

## 🚀 Key Highlights
- Automated news scraping from GECA sources  
- Scheduled cron jobs for daily updates  
- Admin authentication using JWT  
- Email notifications for new announcements  
- Smooth vertical scrolling news ticker  
- Responsive UI with animated gradient background
